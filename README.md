# FITLOG Gym Management

Bu proje hazırlanırken yerel server kullanılmıştır. Projeyi kendi bilgisayarınızda çalıştırabilmek için kodu düzenlemeniz gerekmektedir.

Projeyi kendi yerel ortamınızda çalıştırmak için şu adımları izleyin:

Veritabanı Kurulumu:

Database klasörü içindeki FitLogDB.sql dosyasını açın.

SQL Server Management Studio (SSMS) üzerinde bu scripti çalıştırarak tabloları ve örnek verileri oluşturun.

Bağlantı Ayarı:

Visual Studio'da projeyi açın.

Veritabanı bağlantı cümlesindeki (ConnectionString/ConString) Server kısmını kendi SQL Server isminize göre güncelleyin.

Çalıştır:

F5 tuşuna basarak projeyi derleyin ve başlatın.
